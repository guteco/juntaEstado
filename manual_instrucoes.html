🤖 Monitor de Vagas - Junta Médica RN

Este manual guia você desde a instalação do Python até a execução do monitoramento de vagas.

Objetivo: O script verifica automaticamente o site da Junta Médica a cada 10 segundos (Modo Turbo) e avisa no seu celular via Telegram (e com BIPES sonoros no PC 🔊) assim que uma vaga surgir. Ele também manda um sinal de vida ("Heartbeat") a cada 1 hora.

Parte 0: Baixando o Projeto (GitHub) 🐙

1. Download

1. Acesse o repositório no GitHub (link onde você está lendo isso).

2. Clique no botão verde Code -> Download ZIP.

3. Extraia a pasta em um local de fácil acesso (Ex: C:\Projetos\Junta Médica).

Parte 1: Instalando o Python 🐍

1. Baixar e Instalar

1. Acesse o site oficial: python.org/downloads

2. Clique no botão amarelo Download Python.

3. Ao abrir o instalador, MARQUE A CAIXINHA: Add Python to PATH (Isso é muito importante!).

4. Clique em "Install Now" e aguarde finalizar.

2. Instalar Bibliotecas

Abra o seu terminal (Prompt de Comando ou PowerShell) e digite o comando abaixo para instalar a ferramenta de conexão:

pip install requests

Parte 2: Configurando o Telegram 📱

3. Criar o Bot (Obter Token)

1. Abra o Telegram e pesquise por @BotFather.

2. Clique em Começar e envie o comando: /newbot

3. Digite um nome para o bot (Ex: Monitor Junta Médica).

4. Digite um username único que termine em "bot" (Ex: JuntaMonitor_bot).

5. O BotFather vai te dar um código longo vermelho. Esse é o seu TOKEN.

Exemplo genérico: 123456789:ABCDefGhIjkLmNoPqRsTuVwXyZ...

4. Obter seu ID de Usuário (Chat ID)

1. Pesquise por seu novo bot no Telegram e clique em Começar (ou envie um "Oi").

2. Agora, acesse o seguinte link no seu navegador (substitua pelo SEU TOKEN):

https://api.telegram.org/botSEU_TOKEN_AQUI/getUpdates

3. Você verá um texto confuso na tela. Procure por "id": 123456789 dentro de "chat".

4. Esse número é o seu CHAT ID.

Parte 3: Configurando e Rodando ⚙️

5. Configurar e Testar

Abra o arquivo monitor_junta.py e coloque suas credenciais.

Para testar o bot, execute: python test_telegram.py

Para inciar o monitoramento, execute: python monitor_junta.py

Como será a notificação? (Exemplo) 🔥

Quando o robô encontrar uma vaga, seu computador emitirá 3 BIPES ALTOS e você receberá instantaneamente algo assim no seu celular:

Monitor Junta Médica
✅ VAGA ENCONTRADA!

📅 Data: 31/01/2026
⏰ Turno: MANHA

🔗 Acesse Agora

❓ Solução de Problemas

  • Erro vermelho (ConnectTimeout / 500): Normal. O site da Junta é instável. O bot vai ignorar e tentar de novo em seguida.
  • Comando 'python' não reconhecido: Você esqueceu de marcar Add Python to PATH na instalação. Reinstale o Python marcando essa opção.

🖥️ Entendendo o Console (O que significam as letras?)

O monitor imprime mensagens rápidas para você saber o que está acontecendo:

  • 🔎 29/01 (TARDE) -> Status: 200 | Vz
    Vz (Vazio): O site respondeu, mas NÃO há vagas. O script continua procurando.
  • ⚠️ Instabilidade (Erro 500)
    Site Lotado: O servidor da Junta travou. O bot ignora e tenta de novo. Não se preocupe.
  • ⚠️ Erro SSL / Timeout
    Site Lento: A conexão demorou demais ou caiu. O bot tenta de novo.
  • ✅ VAGA ENCONTRADA!
    SUCESSO!: O computador vai apitar e você receberá a mensagem no Telegram.
