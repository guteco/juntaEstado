🤖 Robô da Junta Médica (Guia para Iniciantes)

Siga este guia passo-a-passo para instalar e ligar o robô que vai vigiar as vagas para você.

⚠️ IMPORTANTE: Leia tudo com calma. Se pular um passo, não vai funcionar.

PASSO 0: Baixando o Robô 📦

1. Pegando os Arquivos
  1. Acesse o site onde o código está (GitHub).
  2. Procure um botão verde chamado < > Code. Clique nele.
  3. No menu que abrir, clique em Download ZIP.
  4. Vai baixar um arquivo zipado. Extraia ele (tire de dentro do zip) para uma pasta fácil, tipo Meus Documentos ou Área de Trabalho.

PASSO 1: Preparando o Computador 💻

1. Baixe o Python

O robô precisa de um "programa motor" chamado Python para rodar.

  1. Acesse: python.org/downloads
  2. Clique no botão amarelo grande Download Python.
  3. MUITO IMPORTANTE (O PULO DO GATO 😺):
    Quando abrir o instalador, procure uma caixinha lá embaixo chamada "Add Python to PATH". Você TEM que marcar ela. Se não marcar, nada funciona.
  4. Depois clique em Install Now e espere terminar.

PASSO 2: Instalando as Ferramentas 🔧

2. A Tela Preta (Terminal)

Agora vamos instalar a "peça" que conecta na internet.

  1. No seu computador, aperte a tecla do Windows e digite: cmd
  2. Vai aparecer um programa preto chamado "Prompt de Comando". Abra ele.
  3. Nessa tela preta, digite exatamente isso e aperte ENTER:

    4. pip install requests

  4. Se aparecer um monte de texto branco e terminar com "Sucessfully installed", deu certo! 🎉

PASSO 3: Configurando o Robô 🤖

3. Colocando suas Senhas

Você baixou uma pasta cheia de arquivos, certo? Vamos mexer neles.

  1. Entre na pasta do projeto.
  2. Procure o arquivo monitor_junta.py.
  3. Clique com o botão direito nele -> Abrir com -> Bloco de Notas.
  4. Encontre o local certo: Role um pouco para baixo até ver o título gigante:

    5. # CONFIGURAÇÕES -- (PRESERVADAS)

  5. Coloque suas Senhas: Logo abaixo desse título, você verá TELEGRAM_BOT_TOKEN e TELEGRAM_CHAT_ID. Mude APENAS ali.
  6. (Opcional) Ajuste a Velocidade: Também nessa mesma parte, você pode ajustar o "Modo Humano":
    • INTERVALO_MIN = 3.0 (Mínimo)
    • INTERVALO_MAX = 8.0 (Máximo)
  7. Salve o arquivo (Arquivo -> Salvar) e feche.

PASSO 3.5: Teste Antes de Rodar 🧪

Teste Rápido

Para ter certeza que seu Telegram está funcioando, criamos um arquivo de teste que já pega sua senha automaticamente.

  1. Na tela preta (terminal), digite:

    2. python test_telegram.py

  2. Se o seu celular apitar com uma mensagem do robô, SUCESSO! 🎉
  3. Se der erro vermelho, você colou o Token ou ID errado no passo anterior. Revise.

PASSO 4: Ligando o Robô 🚀

4. Hora da Verdade
  1. Abra a pasta do projeto (onde está o arquivo do passo anterior).
  2. Clique na barra de endereço lá em cima (onde diz o caminho da pasta, tipo C:\Users...) e apague tudo.
  3. Digite cmd ali mesmo e dê ENTER. A tela preta vai abrir de novo.
  4. Agora digite o comando mágico para ligar o robô:

    5. python monitor_junta.py

  5. Se ele escrever "🤖 Monitor Iniciado" e começar a aparecer datas... PARABÉNS! Está rodando. 👏
🔊 Dica: Deixe o volume alto! O computador vai apitar 3 vezes quando achar a vaga.

❓ O que significam as mensagens na tela?

  • 🔎 20/01 a 19/02 -> Vz = Vazio. O robô olhou e não tinha vaga. Ele vai continuar tentando sozinho.
  • ⚠️ Instabilidade (Erro 500) = Site Cheio. O site da Junta travou porque tem muita gente acessando. O robô sabe lidar com isso, ele espera e tenta de novo. Relaxa.
  • ✅ VAGA ENCONTRADA! = CORRE!!! Vai abrir o site e agendar agora!
